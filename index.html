¿Qué es HTML?

HTML, o Lenguaje de Marcado de Hipertexto, es el estándar para crear páginas web. Es el esqueleto de cualquier sitio web.

Estructura básica

Una página HTML se compone de elementos como <html>, <head> y <body>.

Etiquetas principales

HTML utiliza diferentes etiquetas para definir el contenido y estructura de una página web. A continuación, algunas etiquetas principales y sus funciones:

Etiqueta Función Ejemplo
<h1> - <h6> Define encabezados de diferentes niveles. <h1>Título Principal</h1>
<p> Crea un párrafo de texto. <p>Este es un párrafo.</p>
<a> Crea un enlace a otra página o recurso. <a href="https://example.com">Visitar</a>
<img> Inserta una imagen. <img src="imagen.png" alt="Texto alternativo">
<ul> - <ol> Crea listas desordenadas y ordenadas respectivamente. <ul><li>Ítem</li></ul>
<div> Define un contenedor genérico para agrupar contenido. <div>Contenido aquí</div>
<span> Aplica estilos o agrupa texto dentro de una línea. <span style="color:red">Texto en rojo</span>
<table>, <tr>, <td>, <th> Crea tablas, filas y celdas para organizar datos. <table><tr><td>Dato</td></tr></table>
<form> Crea un formulario para recolectar datos del usuario. <form><input type="text"></form>
<input> Crea campos interactivos para entrada de datos. <input type="text" placeholder="Escribe aquí">
<button> Crea un botón interactivo. <button>Enviar</button>
<video> Inserta un video con controles. <video src="video.mp4" controls></video>
<audio> Inserta un audio con controles. <audio src="audio.mp3" controls></audio>
<header> Define la cabecera de una sección o página. <header>Encabezado</header>
<footer> Define el pie de página. <footer>Pie de página</footer>
<article> Define contenido independiente como artículos. <article>Artículo aquí</article>
<section> Define una sección temática dentro de la página. <section>Sección aquí</section>
