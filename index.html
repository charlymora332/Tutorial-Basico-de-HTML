¿Qué es HTML?

HTML, o Lenguaje de Marcado de Hipertexto, es el estándar para crear páginas web. Es el esqueleto de cualquier sitio web.

Estructura básica

Una página HTML se compone de elementos como <html>, <head> y <body>.

Etiquetas principales

HTML utiliza diferentes etiquetas para definir el contenido y estructura de una página web. A continuación, algunas etiquetas principales y sus funciones:

Etiqueta Función Ejemplo
<h1> - <h6> Define encabezados de diferentes niveles. <h1>Título Principal</h1>
<p> Crea un párrafo de texto. <p>Este es un párrafo.</p>
<a> Crea un enlace a otra página o recurso. <a href="https://example.com">Visitar</a>
